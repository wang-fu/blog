## 被 Cursor "背刺"

一个月前，Cursor 按次付费绝版，官方全部改成按量计费。

这对我来说是致命的。

我的工作习惯是在一个窗口内持续工作很久，上下文越来越大，token 消耗飞快。原本 20 刀的额度，一天不到就用完了。也就是说，充一个 Cursor Pro 会员，1-2 天就烧没了。而之前的按次套餐，1000-1500 次能用半个月到一个月（羊毛版）。

成本直接飙升了 10 倍不止。被迫转移阵地。

## Claude Code CLI：能力顶级，体验 Geek

Claude Code 很早就出现了，但我最初的体验并不好——终端交互，回撤、滚动不方便，插件体验也很粗糙。对我这种追求极致视觉的人而言，眼睛受不了，效率也打折。

但我发现另一个羊毛真相：某些 Claude Code 的拼车套餐，量大管饱。

同样是调用 Claude Sonnet/Opus 4，价格不到 Cursor 的 1/10。几百块就能随心所欲地用。比我之前极端情况连续一个月付 4-5 个 Cursor 会员便宜爆了。

于是我忍着用了几天 CLI。

期间还遇到一件事——公司里不少产品、设计同事也想用 Claude Code（我安利过能写 PPT），结果卡在了安装上。什么是 npm？Node.js 怎么装？有人捣鼓了好几天。这很正常，对非技术人员来说，这些东西就是天书。

还有一个更私人的痛点：公司开会太多了。

经常是一些无关紧要、甚至和自己完全无关的会议，被拉进去坐着。白天全在开会，晚上才能指挥 AI 干活。

我就在想：能不能做一个东西，一键安装、小白可用，还能让我在会议室里掏出手机、笔记本，就能远程操控家里或公司电脑上的 Claude Code？

## 于是我做了 Halo

需求很简单：
- 基于 Claude Code SDK，保留 cc 100% 的 Agent 能力
- 可视化界面，不用再盯着终端
- 一键安装，不需要用户懂 Node.js 和 npm，小白可用
- 支持远程控制——任何设备，任何网络，随时能用

第一个版本花了几个小时搭起来。关键是打包之后，剩下的所有功能，100% 由 Halo 自己完成。

我再也没有打开过 Claude Code CLI 和 Cursor。

| 能力 | Claude Code CLI | Halo |
|--|--|--|
| Agent 能力 | ✅ 完整 | ✅ 完整（同一引擎）|
| 界面 | ❌ 仅终端，无 GUI | ✅ 可视化界面 |
| 远程访问 | ❌ 不支持 | ✅ 任意设备远程 |
| 安装门槛 | ❌ 需要 Node.js、npm | ✅ 一键安装 |
| 产物管理 | ❌ cd 命令 | ✅ 右侧产物栏，一目了然 |
| 文件预览 | ❌ 需要手动打开 | ✅ v2 内置预览，支持代码高亮 |
| AI 浏览器 | ❌ 复杂 MCP 配置 | ✅ v2 内置 |
| 适合人群 | 技术人员 | 所有人 |

同样的底层能力，完全不同的使用体验。

## 这三周发生了什么

100% 的代码都是 Halo 写的。

不是夸张，是事实。我只负责描述需求、看结果、给反馈。从内部日常需求，大到类似 VS Code 规模的大型 IDE 项目，全程 Halo 主导。

最科幻的一次体验：昨天我让 Halo 给自己开发一个「AI 浏览器」功能——内置 Chrome 同款内核，让 AI 能直接操作网页，帮我完成购物、填表、比价这些事。甚至体验感觉比 openai 的 AI 浏览器 atlas 还快（多亏 claude code 底层支持）。

这个功能从 0 到完成，Halo 自己写的，核心人工投入不超过 3 小时。

我还把 Halo 给了身边几个完全不会写代码的同学试用。其中一个财务同学，用它从零搭建了完整的前后端，还自己部署到了服务器上。

这让我更加确信一件事：顶级 Agent 的能力早就到了，但复杂的交互把普通人拒之门外。

## 代价：7×24 小时的牛马枷锁

有一天我在医院打针（上上周六），病床上躺着，手机连上家里电脑的 Halo，让 AI 跑了一下午的数据处理任务。

打完针回家，活干完了。

那一刻我意识到，我给自己造了一副枷锁。

以前的借口是「电脑不在身边」「环境没配好」「晚点再说」。

现在没有借口了。随时随地，掏出手机，AI 就能帮你干活。

牛马人给牛马人自己创造的 7×24 小时工作系统。

这是效率革命，也是一种诅咒。

这让我开始思考：AI 会不会是一种被资本异化的"生产力"，AI 提高的效率到底是给我们带来了幸福还是"枷锁"？

在场的同学们普遍认可，AI 提效更多的是被掌握 AI 生产资源的"资本"所利用，幸福并不会因技术的进步而到来。但一个基本的事实是，在资源竞争下，未来普通人之间的较量就是 AI 生产力工具，就如同会用计算机的财务和手写记账的财务一样。

想到这里，最终还是底层牛马的相互伤害，不免有些悲伤。

## 最后

如果你喜欢 Claude Code 的能力，应该会喜欢 Halo。但 Halo 不是 IDE、不是 CLI，它丢弃复杂的交互，只是把最尖端的 Agent（CC）能力，封装进符合人类直觉的界面里，同时赋予随时随地用自然语言访问它的远程能力。

任何人都可以下载、安装、使用。不需要懂编程，不需要配环境，双击就能跑。

👉 下载地址：https://hello-halo.cc

macOS / Windows 都支持。安卓、iOS 可通过 web 远程访问。

「把 Claude Code 装进口袋」—— 这就是 Halo。
