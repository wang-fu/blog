<div class="content"><p>最近 OpenClaw 突然火了，腾讯被指责为抄袭 OpenClaw，并回应是「标注 clawhub 为原始来源」</p><p>作为它老东家出来的人，看着这件事发酵，心情挺复杂的。有些话，我觉得有必要说。</p><br><hr><br><p><strong>先说 OpenClaw</strong></p><br><p>客观讲，OpenClaw 的热度确实有炒作成分。远程控制、定时任务、Agent 能力，这些东西在极客圈早就不新鲜了。</p><br><p>但它做对了一件事：把这些能力第一次大规模地做成了面向普通用户的产品，让 Agent 自动化这类工具真正进入了大众视野。这一点，值得肯定。</p><br><p>至于腾讯做大陆镜像这件事，我认为没什么好批评的。国内源本来就是正常的基础设施操作，Node.js、Python 都有国内源，没有这些镜像普通用户根本用不了。ClawHub 的 skill 大多来自开源仓库，版权上也站得住脚。这件事腾讯的回应是客观的。</p><br><hr><br><p><strong>但腾讯的问题，不在这里</strong></p><br><p>真正让我失望的，是腾讯在 AI 领域一贯的姿态：<strong>跟随，而不是思考。</strong></p><br><p>Anthropic 发布协作功能 Cowork，腾讯随即发布"Code Buddy Cowork"，连名字都直接借用，还高调官宣。OpenClaw 火了，内部 QClaw、XClaw 各种类似产品纷纷冒出，在公众号一个接一个地宣发。</p><br><p>我不是说学习竞品有问题。问题是：<strong>为什么要等别人先做，你才知道这是个需求？</strong></p><br><p>半年前，Claude Code 已经很火了。腾讯是国内 C 端用户基数最大的公司之一，拥有海量资源。可有没有人认真想过：怎么把顶尖的 Agent 能力交给普通人用？做一个小白化、可视化、一键安装的桌面 Agent？</p><br><p>这是个明摆着的巨大市场和痛点。但他们没有。</p><br><p>OpenClaw 的核心场景——"在火车上、开会时，用手机远程指挥桌面帮你干活"——这种需求，半年前就已经清晰存在于市场里了。谁没有过这种临时的念头？这些再明显不过的痛点，腾讯视而不见，等 OpenClaw 火了再跟进。</p><br><p>这不是一两次巧合，是<strong>系统性的跟随主义</strong>。</p><br><hr><br><p>说一件可能让你意外的事。</p><br><p>半年前，我被朋友连续问烂了同一个问题："怎么装 Claude Code？能不能随时随地用？"</p><br><p>烦了，于是花了一个周末，自己动手——把 Claude Code 完整地产品化：手机里7×24 小时指挥你的电脑Agent，驱动 AI 浏览器自主完成任务，配上友好的可视化界面与完整的文件管理能力。</p><br><p>写代码、写 PPT、做调查、写报告、部署服务器、整理桌面、操控浏览器……凡是 Agent 能力所及，一概支持。<strong>开源，免费，一键安装。</strong></p><br><p>这就是 Halo。</p><br><p>没有大 V 推荐，没有营销投入，靠用户口碑慢慢传开。目前下载超 3 万，周活跃超一千。感谢一路支持的朋友们，也特别感谢网友 mr 的贡献。</p><br><p>最新版本已支持 7×24 小时数字人——你可以在数字人商店雇一个自动发小红书、回复 B 站评论的机器人，也可以直接让 AI 帮你创建一个专属的数字人。</p><br><p><strong>把 Cowork 的可视化能力和 OpenClaw 的远程自动化等能力结合在一起的产品，本应该诞生在腾讯这样拥有海量 C 端用户和充裕资源的公司里。但它没有。它出现在了一个周末，出现在了一个被朋友问烦了的开发者手里。</strong></p><br><hr><br><p><strong>最后</strong></p><br><p>在 AI 时代，人人都是开发者。只要你有想法，有创意，Halo 都能帮你实现。</p><br><p>我们相信，未来的计算平台应该是一个标准化的 Agent 运行时，让 AI 智能体能够持久化、自主化地接管复杂业务流。这一切应该是开源的、可共建的，而不是被商业公司垄断的。</p><br><p>最新版本已接入 Qwen、GLM、Kimi 等国产开源模型，免费可用。同时接入了 Kiro，客户端直接登录 GitHub 账户，无需魔法，无需中转，直连免费 Claude 模型。</p><br><p>欢迎 Star，欢迎体验，欢迎一起共建。</p><p>https://github.com/openkursar/hello-halo</p><br><p>官方文档：</p><p>https://hello-halo.cc/docs/guide/what-is-halo</p><br><hr></div>
